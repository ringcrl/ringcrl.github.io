横屏 H5 有些坑。

解决方法

有无解锁竖屏锁定，手机横向时可以浏览。

横竖屏适配

1
2

transform-origin: 0 0 0;
transform: rotate(90deg) translateY(-clientWidth);

使用 API 来判断屏幕的横竖屏情况

// 返回一个 match 的情况，如果是横屏就不处理，如果是竖屏，就采用上一步的【横竖屏适配】方案
